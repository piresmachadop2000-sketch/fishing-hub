# Fishing Hub

Marketplace de pesca esportiva no Brasil — MVP visual completo em Next.js pronto para deploy na Vercel.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Deploy na Vercel

1. Suba este projeto para um repositório no GitHub.
2. Entre na Vercel.
3. Clique em **Add New Project**.
4. Importe o repositório.
5. Framework: **Next.js**.
6. Build command: `npm run build`.
7. Output: padrão da Vercel.
8. Clique em **Deploy**.

## Telas incluídas

- Landing page
- Login / escolha de perfil
- Cadastro de pescador
- Cadastro de guia
- Home do pescador
- Busca com filtros
- Página de experiência
- Perfil do guia
- Fluxo de reserva
- Minhas reservas
- Chat
- Avaliações
- Dashboard do guia
- Criar anúncio
- Agenda do guia
- Financeiro do guia
- Admin interno
