# Fishing Hub

Marketplace responsivo de pesca esportiva no Brasil, conectando pescadores a guias verificados.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide React

## Como rodar

```bash
npm install
npm run build
npm run dev
```

Abra `http://localhost:3000`.

## Deploy na Vercel

1. Suba este projeto para um repositório Git.
2. Importe o repositório na Vercel.
3. Use as configurações padrão de Next.js.
4. Clique em Deploy.
